# Althingi Investigation and Probing

## About This Project

This is an assignment to discover externally exposed infrastructure and services for Alþingi, Iceland's parliament. The goal is to find publicly accessible systems, subdomains, and potential security issues to help improve their cybersecurity.

This project will contain txt files, screen shots (png), tools and commands used for reconnaissance.
